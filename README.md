# Weather Application

A simple weather application built with Node.js, Express, Axios, and Weather API. The app allows users to enter a city name and get the current weather information for that city.

## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- Axios
- HTML/CSS
- JavaScript

  ## Features

- Fetch current weather data for any city using the Weather API.
- Store and periodically update weather data in a MongoDB database.
 - Simple frontend interface.

  ## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/gaurav637/Weather-Application
    ```

2. Navigate to the project directory:
    ```bash
    cd weather
    ```

3. Install dependencies:
    ```bash
    npm install
    ```

4. Create a `.env` file in the root directory and add your MongoDB URI and Weather API key:
    ```env
    MONGO_URI=your_mongodb_uri
    PORT=server port
    ```

5. Start the server:
    ```bash
    npm start
    ```

## Prerequisites

Before you begin, ensure you have met the following requirements:

- Node.js and npm installed on your machine.
- An weather api  API key. You can get one by signing up at [weather api](https://www.weatherapi.com/api).
